# Movie Genre Prediction with Bert

An example of multi label classification using Pytorch Lightning and Bert.

## Requirements

    - conda

## Installation

```sh
conda env create --name <envname> -f environments.yml
conda activate <envname>
```

## Training
```sh
python train.py
```

## Prediction
```sh
python predict.py
```
